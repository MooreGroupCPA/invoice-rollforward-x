Information Requests Received Review

Upload the Information Requests DOCX (EZ Form or SOC report testing matrices) and the Evidence ZIP. This tool finds rows where the Test Results Determined by the Service Auditor cell contains “Provide by”, then checks if the evidence ZIP includes files whose names contain matching numeric Control Points (e.g., 1.2, 6.18). SOC 2 refs like CC6.4 are ignored.

This is the client’s EZ Form (SOC 2) or the SOC 1 report where info requests appear in the testing matrices.
Filenames can include multiple control points, e.g. “1.2, 1.4, 1.6 Employee Handbook”.
